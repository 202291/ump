```bash
npm i @yetnt/ump
```
```js
const Ump = require(@yetnt/ump)
```

## Features
What can this do?

### Prime Factorization

```js
const Ump = require(@yetnt/ump)

Ump.primeFactorization(50) // [2, 5, 5]
Ump.primeFactorization(5000) // [2, 2, 2, 5, 5, 5, 5]
Ump.primeFactorization(93928893) // [3, 17, 103, 17881]
Ump.primeFactorization(3) // [3]
```

### Ratio(s)

```js
const Ump = require(@yetnt/ump)

Ump.DinRatio(500, 3, 2) // [300, 200]
```
